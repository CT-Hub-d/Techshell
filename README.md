# Techshell
a unix-like shell that operates simmilar to a bash
Group: Collin Tucker(Just me)
Responsibilities: I worked on all of it
Description: Its a unix like shell that reads, parses and then exectues those commands using fork() and execp(). it can change directory and exit. 


to clone: git clone https://github.com/CT-Hub-d/Techshell.git

to compile: gcc techshell.c -o techshell

to run: ./techshell

you need a Unix/Linux based system, a GCC compiler and C libaries
